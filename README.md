This project is an Explainable AI (XAI) diagnostic tool designed for the multi-class classification of skin lesions using the HAM10000 dataset. Built upon a ResNet50 deep learning architecture, the system accurately identifies seven types of skin conditions. To ensure high sensitivity for life-threatening malignancies like Melanoma, strategic data oversampling was implemented to resolve severe class imbalance. Crucially, the framework integrates Grad-CAM to generate visual heatmaps, transforming opaque 'black-box' predictions into transparent, clinically interpretable evidence. The entire pipeline is deployed as an interactive Streamlit web application, providing medical professionals with real-time, trustworthy diagnostic support.
